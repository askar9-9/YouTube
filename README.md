# YouTube Clone

**Пет-проект**: Клон платформы для видео-хостинга YouTube. Этот проект реализует основные функции YouTube, такие как регистрация, аутентификация, загрузка видео, поиск, фильтрация контента, а также дополнительные возможности, включая стриминг видео и подключение через **OAuth**.

Проект был создан в качестве учебной практики для улучшения навыков работы с Go, React и современными технологиями.

## Цели проекта

- Изучение разработки с использованием **Go** для бэкенда и **React** для фронтенда.
- Реализация основных функций видео-платформы.
- Изучение работы с базами данных, такими как **PostgreSQL** и **MongoDB**.
- Применение контейнеризации с **Docker** и **Docker Compose**.
- Разработка и настройка CI/CD пайплайнов с использованием **GitHub Actions**.

## Функции проекта

### Основные функции:
1. **Регистрация и аутентификация**
   - Регистрация новых пользователей и верификация email.
   - Аутентификация через **JWT токены** для безопасности сессий.
   - **OAuth-аутентификация** с поддержкой популярных сервисов, таких как Google, Facebook и GitHub для быстрого подключения.

2. **Роли и права доступа (RBAC)**
   - Разграничение прав пользователей (администраторы, обычные пользователи, модераторы).

3. **Уведомления**
   - Пользователи получают уведомления о новых видео, комментариях и подписках.

4. **Поиск и фильтрация**
   - Удобный поиск видео по ключевым словам.
   - Фильтрация контента по категориям, дате и популярности.
   - Поддержка пагинации и сортировки.

5. **Стриминг видео**
   - Прямые трансляции и стриминг видео с возможностью чата.

6. **API-интеграции**
   - Открытое API для сторонних сервисов и интеграций.

### Дополнительные возможности:
1. **Мультиязычность (i18n)**
   - Поддержка нескольких языков интерфейса.

2. **CI/CD и DevOps**
   - Автоматизация процессов тестирования и деплоя с использованием **GitHub Actions**.

3. **Запланированные задачи (Scheduled Worker)**
   - Обработка фоновых задач, таких как отправка уведомлений и генерация отчетов.

## Технологический стек

- **Frontend**: React, Redux
- **Backend**: Go (net/http, JWT, OAuth)
- **Базы данных**:
  - **PostgreSQL**: Для хранения данных пользователей и видео.
  - **MongoDB**: Для хранения метаданных и комментариев.
  - **Redis**: Для кэширования данных.
- **Контейнеризация**: Docker, Docker Compose
- **Message Broker**: Apache Kafka для обработки событий и асинхронных задач.
- **CI/CD**: GitHub Actions для автоматической сборки и деплоя.
